Unidades de medidas

Dentro do universo das unidades de medida, cada uma se adapta de forma mais eficaz a diferentes elementos em uma página web.

Para textos temos:

unidades como pixels (px), em e rem são comuns, permitindo um controle preciso sobre o tamanho das fontes.

Para imagens

a unidade percentual (%) é frequentemente preferida, pois se ajusta dinamicamente à largura e à tela do usuário, garantindo uma experiência visualmente agradável em diferentes dispositivos.

Obs:

em e rem são bastante utilizadas para que a página se adapte a tela do usuário, diferente do px que mesmo com a tela do usuário maior ou menor ele vai dar um espaçamento X e fixo

Obs 2:

1em equivale a 16 px, então sé colocarmos um texto com 25px ele vai ficar num tamanho bom, mas se colocarmos 25em ele vai ficar enorme
