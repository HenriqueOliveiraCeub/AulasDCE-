O que é responsividade

Responsividade refere-se à capacidade de um site ou aplicativo web se adaptar e fornecer uma experiência de usuário otimizada em uma variedade de dispositivos e tamanhos de tela. Isso inclui desktops, laptops, tablets e smartphones, bem como diferentes orientações de tela, como retrato e paisagem.

Como fazer imagens ficarem responsivas

Para fazer a imagem se adaptar ao site é muito simples, basta colocar o width dela como 100% e dessa forma ela vai ocupar todo o espaço disponível e se o usuário tiver uma tela menor a imagem vai se adaptar para a tela dele.
