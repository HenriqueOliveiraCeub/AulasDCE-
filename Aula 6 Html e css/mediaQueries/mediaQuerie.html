O que são media queries

(abra o código "MediaQuerie.css" para vizualizar como e onde chamamos elas)
Media queries são um recurso do CSS que permite adaptar o estilo de uma página com base nas características do dispositivo que está sendo usado para visualizá-la, como largura da tela, altura da tela, orientação da tela e tipo de dispositivo.

Com media queries, você pode definir diferentes estilos CSS para diferentes condições. Por exemplo, você pode ter um estilo para telas grandes de desktop e outro estilo para telas menores de dispositivos móveis.

O que são breakpoints

Breakpoints são pontos específicos de largura da tela em que o layout de uma página é ajustado para garantir uma experiência de usuário adequada.

Eles são usados em conjunto com media queries. Você define breakpoints em larguras de tela específicas e, em seguida, usa media queries para aplicar estilos diferentes quando a largura da tela atinge esses pontos.
